# **App Name**: CUET Prep 2026

## Core Features:

- Study Material: Comprehensive modules covering the CUET exam syllabus, with topics like reading comprehension, synonyms, antonyms, idioms, and grammar.
- Adaptive Quizzes: Personalized quizzes that adjust difficulty based on the user's performance to identify strengths and weaknesses. All questions and answers are hardcoded.
- Marking Strategy Guide: Tool that offers recommendations on whether to attempt, skip, or review questions based on confidence levels during mock tests, simulating exam-day scenarios. All recommendations are based on hardcoded logic.
- Progress Tracking: Visual dashboards displaying the user's progress, quiz scores, and areas needing improvement.
- Exam Strategy Guide: Detailed guides covering exam-taking strategies, time management techniques, and section-specific approaches for CUET 2026.
- Doubt Resolution: Provides hardcoded answers to user queries based on the study material, exam patterns, and general preparation tips.

## Style Guidelines:

- Primary color: Light lavender (#E6E6FA) for a calming and intellectual feel, aligning with study and focus.
- Background color: Off-white (#F8F8FF), providing a clean and distraction-free reading experience.
- Accent color: Pale blue (#ADD8E6) to highlight important information and CTAs, promoting a sense of trust and clarity.
- Body and headline font: 'Inter', a sans-serif font, will be used for its clean, modern, objective look; appropriate for both headlines and body text.
- Code font: 'Source Code Pro' for any code snippets that are part of study guides, since it is easily legible.
- Use clear, minimalist icons to represent different modules and topics.
- A clean, structured layout to ensure ease of navigation.
- Subtle animations on quiz submission.
